Quick Docs App

App.js:
Token based Automatic Logging in using Backend JWT Token Verification

Components:

1. Customized Alert Boxes (Error, Success, Warning, Permission Required)
2. Customized Header Component

1. Login Screen:
Completely Validated with 2 Input fields: username and password and Restricted api hit only when all check parameters are passed.

2. Signup Screen:
User must upload Profile Picture, Enter Full name, valid and new username (not a pre existing one, if does app will throw a warning alert: username exists), Password (must be min 8 len, min 1 capital, min 1 small, min 1 special character, min 1 number, else throws alert boxes)
